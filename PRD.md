# Product Requirements Document (PRD) - 검 강화 게임

## 1. 개요
"검 강화 게임"은 사용자가 검을 강화하고, 강화 성공 시 보상을 얻을 수 있는 모바일 웹 기반 게임입니다. 사용자는 출석체크를 통해 골드를 획득하고, 골드를 사용하여 검을 강화하거나 판매할 수 있습니다.

## 2. 타겟 사용자
- 모바일 게임을 즐기는 10대~30대
- 간단한 RPG/강화 요소가 있는 게임을 선호하는 사용자
- 짧은 시간 동안 즐길 수 있는 캐주얼 게임을 선호하는 사용자

## 3. 주요 기능

### 3.1 사용자 인증
- 전화번호 기반 회원가입/로그인
- 닉네임 설정 (한글, 영문, 숫자만 가능, 최대 15자)
- 자동 로그인 유지

### 3.2 메인 게임
- **검 강화 시스템**
  - 강화 단계별 확률 및 비용 (1~20강)
    | 현재 레벨 | 강화 비용 | 성공 확률 | 유지 확률 | 하락 확률 | 파괴 확률 |
    |----------|----------|----------|----------|----------|----------|
    | 0 → 1    | 100G     | 95%      | 5%       | 0%       | 0%       |
    | 1 → 2    | 200G     | 90%      | 10%      | 0%       | 0%       |

    | 6 → 7    | 6,400G   | 65%      | 15%      | 15%      | 5%       |
    | 7 → 8    | 12,800G  | 60%      | 10%      | 20%      | 10%      |
    | 8 → 9    | 25,600G  | 55%      | 5%       | 25%      | 15%      |
    | 9 → 10   | 51,200G  | 50%      | 0%       | 30%      | 20%      |
    | 10 → 11  | 102,400G | 45%      | 0%       | 35%      | 20%      |

    | 12 → 13  | 409,600G | 35%      | 0%       | 40%      | 25%      |
    | 13 → 14  | 819,200G | 30%      | 0%       | 45%      | 25%      |
    | 14 → 15  | 1,638,400G| 25%      | 0%       | 45%      | 30%      |

    | 19 → 20  | 52,428,800G| 1%      | 0%       | 49%      | 50%      |
  - 강화 성공 시: 다음 레벨로 상승
  - 유지: 현재 레벨 유지
  - 하락: 1레벨 하락 (단, 0레벨 미만으로는 하락하지 않음)
  - 파괴: 검이 초기화되어 0레벨로 복귀
  - 강화 애니메이션 효과 (성공/실패 시 각기 다른 효과)

- **골드 시스템**
  - 출석체크 보상: 1,000G (1일 1회)
  - 연속 출석 시 추가 보상 (3일 연속: 1,500G, 7일 연속: 3,000G)
  - **검 판매 가격**
    - 1~9레벨: (레벨³ × 50)G (예: 3레벨 = 3³ × 50 = 1,350G)
    - 10~14레벨: (레벨³ × 100)G (예: 10레벨 = 1,000 × 100 = 100,000G)
    - 15~19레벨: (레벨³ × 200)G (예: 15레벨 = 3,375 × 200 = 675,000G)
    - 20레벨: 10,000,000G (고정)
  - 강화 비용: 레벨별 상이 (상단 표 참조)
  - 최대 보유 골드: 99,999,999G

- **아이템 관리**
  - 현재 보유 중인 검 상태 확인

### 3.3 화폐 시스템
- **골드(G)**
  - 게임 내 기본 화폐
  - 출석체크, 검 판매, 미니게임 등을 통해 획득 가능
  - 강화 비용 지불에 사용
  - 최대 보유량: 99,999,999G

- **머니(M)**
  - 프리미엄 화폐 (유료 결제 또는 이벤트 보상으로 획득)
  - 1M = 1,500G (고정 환율)
  - 특별한 아이템 구매에 사용
  - 최대 보유량: 999,999M
  - 환전소에서 골드로 전환 가능 (1M → 1,500G)

### 3.4 환전소
- **기능**
  - 머니(M)를 골드(G)로 전환 (1M = 1,500G)
  - 즉시 환전 가능 (수수료 없음)
  - 일일 환전 한도: 1,000,000G (약 667M)
  - 환전 내역 조회
  - 환전 예상 금액 확인
  - 최소 환전 단위: 1M
  - 최대 환전 단위: 667M (1,000,000G 상당)
  - 24시간 이내 환전 취소 가능

### 3.5 어드민 전용 기능
- **화폐 관리**
  - 특정 사용자의 골드/머니 조회 및 수정
  - 일괄 지급/차감 기능
  - 화폐 변동 내역 로깅

- **강화 확률 조정**
  - 레벨별 성공/하락/파괴 확률 실시간 조정
  - 즉시 적용 가능

- **사용자 제재**
  - 계정 정지/경고
  - 부정 사용자 제재 내역 관리

### 3.5 랭킹 시스템
- 사용자별 검 레벨 기반 랭킹 표시
- 상위 랭커 보상 시스템

### 3.6 관리자 기능
  - 사용자 관리 (정지, 경고, 화폐 조정)
  - 게임 밸런스 조정 (확률, 가격 등)
  - 이벤트 관리 (출석 이벤트, 프로모션)
  - 시스템 알림 발송
  - 데이터 백업 및 복구

## 4. 기술 스택
- **프론트엔드**: HTML5, CSS3, JavaScript
- **스타일링**: Tailwind CSS
- **백엔드**: Supabase (인증, 데이터베이스, 스토리지)
- **빌드 도구**: Vite
- **호스팅**: Vercel 또는 Netlify (추정)

## 5. UI/UX 요구사항
- 모바일 최적화된 반응형 디자인
- 직관적인 인터페이스
- 즉각적인 피드백 (성공/실패 애니메이션)
- 진행 상황 자동 저장

## 6. 보안 요구사항
- 사용자 인증 정보 암호화 저장
- 클라이언트 측 검증 외 서버 측 검증 추가 필요
- API 키 보호 필요 (현재 프론트엔드에 노출됨)

이 PRD는 현재 구현된 기능을 기반으로 작성되었으며, 향후 사용자 피드백과 시장 반응에 따라 지속적으로 업데이트될 예정입니다.
